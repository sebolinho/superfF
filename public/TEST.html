API Documentation

Filmeapi.pro API give you the lowdown on all the methods, request formats, and both required and optional parameters.

1. Movie Embed URL

Use IMDB ID for movie ID

Endpoint

            https://filmesapi.pro/embed/{imdb_id}

2. TV Show Embed URL

Use TMDB or IMDB ID for TV Show ID

Endpoint

            https://filmesapi.pro/embed/{tmdb_id}/{season_number}/{episode_number}

3. List All Movies & Series

{type} required - "tv" or "movie"

Endpoint

            https://filmesapi.pro/list/{type}.json

Examples

https://filmesapi.pro/list/movie.json

https://filmesapi.pro/list/tv.json
